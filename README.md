## Project setup

### Backend
```
cd backend
```
```
pip install -r requirements.txt
```
```
python manage.py makemigrations
```
```
python manage.py migrate
```
```
python manage.py runserver
```

### Frontend
```
cd frontend
```
```
npm install
```
```
npm run dev
```
